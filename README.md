# POS System

A simple Point of Sale system built with React and Vite for bars, restaurants, or cafes.

## Features

- FOH seating layout with interactive tables
- Menu display with items and prices (shown only after selecting a table)
- Add items to cart and submit orders per table
- Ability to cancel a table selection and return to layout
- Calculate total price

## Getting Started

1. Install Node.js if not already installed.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
4. Open your browser to `http://localhost:5173` to view the app.

## Usage

- Browse the menu and click "Add to Cart" for items.
- View the cart and total at the bottom.

## Troubleshooting

- If `npm` is not found, ensure Node.js is installed and added to your PATH.
- Restart your terminal or VS Code after installing Node.js.
